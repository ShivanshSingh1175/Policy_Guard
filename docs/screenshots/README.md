# Screenshots Directory

## How to Add Screenshots

1. Take screenshots of your running application:
   - **login.png** - Login page with Google Sign-In button
   - **dashboard.png** - Dashboard with metrics and charts
   - **policies.png** - Policies & Rules management page
   - **scans.png** - Scans page with history
   - **violations.png** - Violations list with filters
   - **analytics.png** - Analytics page with trend charts

2. Save them in this directory (`docs/screenshots/`)

3. Screenshots are automatically referenced in the main README.md

## Screenshot Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: 1920x1080 or higher
- **Quality**: High quality, clear text
- **Content**: Show actual data, not empty states
- **Browser**: Hide browser chrome if possible (F11 fullscreen)
